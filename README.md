# Saludex
Saludex es una pagina que se enfoca en la salud de sus clientes, enfocándose en prevenir y/o evitar que se pueda llegar a una obtención de enfermedades como: Obesidad, Bulimia, Anorexia, etc. Tambien se enfoca en tratamiento fisico con fisioterapia y acondicionamiento fisico con profesionales capacitados para cada tipo de situación por la que este pasando el cliente. Usando no solo el entrenamiento fisico, sino que tambien masajes, entrenamientos mentales y todo tipo de experiencias que ayuden al cliente a superar su discapacidad
